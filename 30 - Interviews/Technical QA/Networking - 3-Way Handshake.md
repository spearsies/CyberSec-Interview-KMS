# Describe the TCP 3-Way Handshake

## Sample Response
The TCP 3-way handshake is the process used to establish a reliable connection. It consists of:
1. **SYN**: Client sends a synchronize packet to the server.
2. **SYN-ACK**: Server responds with a synchronize-acknowledgment.
3. **ACK**: Client sends an acknowledgment back.
